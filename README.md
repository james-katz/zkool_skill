# Zkool Wallet Skill for OpenClaw

This is a skill designed to be used with the OpenClaw assistant. It allows you to interact with a Zcash wallet using the Zkool GraphQL API.

## How to Get the Skill

To use this skill, you can add it to your OpenClaw skills directory.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/james-katz/zkool_skill.git
    ```

2.  **Place the skill in your skills directory:**
    Move or link the `zkool_skill` directory into your OpenClaw skills folder.

For detailed setup and usage instructions, please refer to the `SKILL.md` file.
